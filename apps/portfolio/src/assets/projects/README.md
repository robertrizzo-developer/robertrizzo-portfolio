# Project Assets

Add screenshots and architecture diagrams here for each project.

## Expected files per project

Each folder should contain:

- `screenshot.png` — Browser screenshot of the live application
- `architecture.png` — Architecture diagram (use Excalidraw or Mermaid)

## How to create architecture diagrams

- **Excalidraw** (excalidraw.com) — Hand-drawn style, export as PNG
- **Mermaid** (mermaid.live) — Code-based diagrams, export as PNG

## Folder structure

```
projects/
├── booking-api/
│   ├── architecture.png    # Full system architecture diagram
│   ├── swagger.png         # Screenshot of Swagger UI
│   └── tests.png           # Screenshot of passing test suite
├── fishing-club/
│   ├── screenshot.png      # Browser screenshot of live site
│   └── architecture.png    # Simple system diagram
├── jarnvilja/
│   ├── screenshot.png      # Browser screenshot of the app
│   └── architecture.png    # Frontend → API → Database diagram
├── weather-app/
│   ├── screenshot.png      # Browser screenshot
│   └── architecture.png    # Browser → Weather API diagram
└── ai-chatbot/

    └── architecture.png    # User → App → LLM API diagram
```
